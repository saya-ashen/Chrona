# Chrona Documentation

Chrona is an AI-native task control plane for scheduling work and driving task execution through agents.

**Install:** `npm install -g @chrona-org/cli`

This documentation is split into focused files:

- [Quick Start](./quick-start.md) — Setup, first run, AI backend configuration
- [Roadmap](./roadmap.md) — Product direction and phases
- [Architecture](../architecture.md) — CQRS + Event Sourcing design
- [Data Model](../data-model.md) — Full schema reference
- [API Reference](../api-reference.md) — REST API endpoints

Recommended reading order:
1. Quick Start
2. Architecture
3. API Reference
