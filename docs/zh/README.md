# Chrona 文档

Chrona 是一个面向 AI 时代的任务与排期控制台，目标是把"想法、排期、执行"串成一个连续流程。

**安装：** `npm install -g @chrona-org/cli`

文档按主题拆分：

- [快速开始](./quick-start.md) — 安装、启动、AI 后端配置
- [路线图](./roadmap.md) — 产品方向和阶段规划
- [系统架构](../architecture.md) — CQRS + 事件溯源设计
- [数据模型](../data-model.md) — 完整 schema 参考
- [API 参考](../api-reference.md) — REST API 端点

推荐阅读顺序：
1. 快速开始
2. 系统架构
3. API 参考
