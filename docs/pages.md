## / — 首页重定向

**URL:** `/` 或 `/[lang]`

**行为:** 自动重定向到 `/schedule`（或 `/[lang]/schedule`）。

**查询参数:** 无

## /schedule — 排程驾驶舱

**URL:** `/schedule` | `/[lang]/schedule`

**查询参数:** 无（工作空间 ID 由应用上下文确定）

**功能说明:**

排程驾驶舱是系统的核心规划页面，提供任务时间规划和管理的全景视图。

**显示内容:**

- **时间线视图:** 以甘特图形式展示当天及未来几天的任务排程，每个任务显示为时间条，颜色根据优先级区分
- **任务队列:** 待排程任务列表，可通过拖拽排入时间线
- **任务列表:** 所有已排程和待排程任务的完整列表，支持排序和筛选
- **冲突提示:** 当存在时间冲突时显示警告标识
- **AI 建议面板:** 展示 AI 生成的排程优化建议

**关键组件:**

- `ScheduleTimeline` — 时间线甘特图组件
- `TaskQueue` — 待排程任务队列
- `TaskList` — 任务列表表格
- `ConflictBanner` — 冲突警告横幅
- `AISuggestionPanel` — AI 排程建议面板

**截图描述:** 页面上部为横向时间线甘特图，彩色任务条按时间排列；左下方为待排程任务队列卡片堆叠；右下方为任务列表表格，包含标题、优先级、状态、排程时间等列。

## /inbox — 干预收件箱

**URL:** `/inbox` | `/[lang]/inbox`

**查询参数:** 无

**功能说明:**

干预收件箱展示所有需要人工干预的任务，如等待审批、等待输入、运行失败等。

**显示内容:**

- **干预项列表:** 每个需要干预的任务以卡片形式展示，包含任务标题、干预类型、紧急程度
- **快速操作按钮:** 每张卡片上提供审批/拒绝、提供输入、重新运行等快捷操作
- **分类标签:** 按干预类型分类（等待审批、等待输入、运行失败等）
- **计数徽章:** 各分类的待处理数量

**关键组件:**

- `InboxCard` — 干预项卡片
- `ApprovalActions` — 审批操作按钮组
- `InputDialog` — 输入提供弹窗
- `InboxFilter` — 干预类型筛选器

**截图描述:** 页面以卡片列表形式展示干预项，每张卡片左侧有彩色类型标识条，中间显示任务信息和干预原因，右侧为操作按钮。页面顶部有筛选标签栏。

## /tasks — 任务中心

**URL:** `/tasks` | `/[lang]/tasks`

**查询参数:** 无（筛选通过页面内 UI 控件实现）

**功能说明:**

任务中心提供所有任务的统一管理视图，支持多维度筛选和批量操作。

**显示内容:**

- **任务表格:** 所有任务的完整列表，列包括标题、状态、优先级、截止时间、创建时间等
- **筛选面板:** 按状态、优先级、时间范围等筛选
- **搜索栏:** 按标题关键词搜索
- **创建任务按钮:** 打开新任务创建表单
- **批量操作工具栏:** 选中多个任务后可执行批量操作

**关键组件:**

- `TaskTable` — 任务数据表格
- `TaskFilters` — 筛选面板
- `CreateTaskDialog` — 创建任务弹窗
- `TaskSearchBar` — 搜索栏
- `BulkActions` — 批量操作工具栏

**截图描述:** 页面顶部为搜索栏和筛选按钮行，中间为数据表格占据主要空间，表格支持列排序，行左侧有复选框用于多选。右上角有蓝色"创建任务"按钮。

## /memory — 记忆管理控制台

**URL:** `/memory` | `/[lang]/memory`

**查询参数:** 无

**功能说明:**

- **管理操作:** 编辑、删除记忆条目

**关键组件:**

- `MemoryList` — 记忆条目列表
- `MemoryCard` — 单条记忆卡片
- `MemoryDetail` — 记忆详情展开面板
- `MemoryFilters` — 记忆筛选器

**截图描述:** 页面以纵向卡片列表展示记忆条目，每张卡片包含记忆摘要、来源任务、创建时间。卡片可点击展开显示完整内容。页面顶部有分类筛选标签。

**关键组件:**

- `WorkspaceHeader` — 工作空间头部信息
- `TaskStatsSummary` — 任务统计摘要
- `RecentTasksList` — 最近任务列表
- `QuickActions` — 快速操作按钮组

**截图描述:** 页面顶部为工作空间名称和描述，下方为统计卡片行（显示待处理、运行中、已完成等数量），再下方为最近更新的任务列表。

## /workspaces/:wId/tasks/:tId — 任务详情

**URL:** `/workspaces/[workspaceId]/tasks/[taskId]` | `/[lang]/workspaces/[workspaceId]/tasks/[taskId]`

**路径参数:**

| 参数 | 说明 |
|------|------|
| workspaceId | 工作空间 ID |
| taskId | 任务 ID |

**查询参数:** 无

**功能说明:**

单个任务的完整详情页。

**显示内容:**

- **任务信息:** 标题、描述、状态、优先级、截止时间等完整信息
- **运行历史:** 该任务的所有运行记录
- **子任务列表:** 子任务及其状态
- **事件时间线:** 任务相关事件的时间轴
- **操作面板:** 编辑、运行、完成、删除等操作按钮

**关键组件:**

- `TaskHeader` — 任务标题和状态头部
- `TaskDetailPanel` — 任务详情面板
- `RunHistory` — 运行历史列表
- `SubtaskList` — 子任务列表
- `EventTimeline` — 事件时间线
- `TaskActions` — 操作按钮组

**截图描述:** 页面左侧为任务详情面板（标题、描述、属性等），右侧为运行历史和事件时间线。底部为子任务列表。顶部操作栏有"运行"、"完成"、"编辑"等按钮。

## /workspaces/:wId/work/:tId — 工作执行页

**URL:** `/workspaces/[workspaceId]/work/[taskId]` | `/[lang]/workspaces/[workspaceId]/work/[taskId]`

**路径参数:**

| 参数 | 说明 |
|------|------|
| workspaceId | 工作空间 ID |
| taskId | 任务 ID |

**查询参数:** 无

**功能说明:**

工作执行页是与 AI 代理交互的主要界面，用于实时监控运行状态、查看输出和与代理沟通。

**显示内容:**

- **运行状态面板:** 当前运行的实时状态（运行中、等待输入、已完成等）
- **输出流:** AI 代理的实时输出内容
- **消息输入框:** 向运行中的代理发送操作员消息
- **输入提供框:** 当代理等待输入时，提供所需输入
- **产物列表:** 代理生成的文件和产物
- **会话历史:** 与代理的交互历史记录

**关键组件:**

- `RunStatusBadge` — 运行状态徽章
- `OutputStream` — 实时输出流面板
- `MessageComposer` — 消息编辑器
- `InputProvider` — 输入提供面板
- `ArtifactList` — 产物列表
- `SessionHistory` — 会话历史

**截图描述:** 页面采用类聊天界面布局。顶部显示任务标题和运行状态徽章。主体区域为输出流，显示代理的工作过程和结果。底部为消息/输入输入框。右侧边栏展示产物列表和任务属性。

## 国际化路由

所有页面均支持 `[lang]` 前缀，当前支持：

| 语言代码 | 语言 |
|----------|------|
| `en` | English |
| `zh` | 中文 |

示例：
- `/en/schedule` — 英文排程页
- `/zh/inbox` — 中文收件箱
- `/en/workspaces/ws_001/work/task_abc123` — 英文工作执行页

不带语言前缀的路由（如 `/schedule`）使用系统默认语言。
